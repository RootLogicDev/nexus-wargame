# ⚡ NEXUS WARGAME v1.0
> **Operation: Zero Day** — A Hardened Offensive Security Lab.

NEXUS is a mobile-friendly, infrastructure-focused CTF designed to take you from a script kiddie to a professional security researcher. 

## 🛠 Features
- **20 Progressive Levels:** From basic enumeration to SUID privilege escalation.
- **Termux Optimized:** Designed specifically for Android/Mobile offensive security.
- **Hardened Engine:** Anti-cheat logic and live session timers.

```bash

bash nexus.sh setup
bash nexus.sh play
